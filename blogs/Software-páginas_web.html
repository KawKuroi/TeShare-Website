¿Que es “Internet”?



         El Internet o Network; término que procede de “red” e “Interconnect” (en la etimologia de la palabra en ingles), por lo que podemos pensar este termino como una gran red que genera una conexion entre usuarios y paginas web a nivel global.


El internet nació para facilitar la comunicación y fuente de información al rededor del mundo, que por consecuente, generó grandes cambios a nivel social, pues se convirtió en el metodo más veloz y eficiente de comunicación personal o masiva de información. De igual manera, este se a convertido en un elemento impresindible en la educación, pues por primera vez en la historia de la humanidad, tenemos todo el conocimiento al alcance de nuestras manos, ¡es más accesible que nunca!



La interconexion global que mecioné anteriormente, surge apartir de una transferencia de hipertextos (enlaces o direcciones que conectan páginas web entre sí) de usuarios a otros por medio de diferentes tipos de estructuras que permiten organizar la información textual o de multimedia, dentro de pequeñas paginas web y a su vez, estas puedan ser compartidas en esta gran red, internet.




¿Como estructuramos esta información?



         Para que una página web pueda presentar la informacion que queremos comunicar, debemos organizarla para que esta sea accesible y consumible, por lo cual, necesitamos un sistema que nos facilite este proceso, lo cual genero el nacimiento de “HTML”.


El HTML (HyperText Markup Language) es un lenguaje que nos permite estructurar nuestra información en una página web, y a su vez, por medio del hipertexto conectar nuestra página web con el internet.




Este lenguaje funciona por medio de etiquetas, las cuales el navegador detecta y luego muestra la información en pantalla, este lenguaje nos permite crear una serie de configuraciones para generar una forma eficiente y atractiva de mostrar esta información; un ejemplo de estas etiquetas podria ser:


<"html">...<"/html"> (Determina el inicio y final de la página web)
<"title">...<"/title"> (Etiqueta utilizada para dar nombre a la página web)
<"img src=”/...”"> (Etiquea utilizada para agregar imagenes)


Este lenguaje es el estandar para la estructuracion de una página web, pero este proceso no queda solo ahí, pues además de estructurar y organizar, podemos decorar cada una de las partes que componen la página web, mediante otros lenguajes como el CSS y no solo eso, tambien podemos convertir nuestra simple pagina web en una aplicacion en si, es decir, que en vez de solo presentar la información para ser consumida, podemos generar una interaccion de la persona que observa la página y la página en si, por medio de diferentes acciones que permiten una funcionalidad adicional a la página, por lenguajes como JavaScript, php, Python.




Ahora que conocemos que es internet, como estructuramos, decoramos y generamos una funcionalidad en el contenido para esta red global…




¿Como podemos compartir nuestra página web?



         Primero, debemos tener capacidad para acceder a internet, por lo tanto, necesitamos un ISP (Internet Service Provider) o proveedor de internet. El proveedor de internet es una empresa encargada de generar una conexión del usuario con internet mediante ciertas tecnologias. El proceso que realiza a nivel basico un proveedor de internet es asignarnos la dirección IP para cada uno de nuestros dispositivos conectables con el mismo


Para poder compartir nuestra página web, primero debemos tener una dirección, pues internet nos permite ser parte de su gran colección de información, pero para que las personas puedan encontrarnos, debemos conocer nuestra direccion IP. Esta dirección es una forma de jerarquia que provee internet para poder identificarnos segun alguno de nuestros dispositivos dentro de esta infinita red, además de conocer donde estan ubicada la información que debemos compartir.




208.65.153.238 (Una de las direcciones IP de Youtube)


Además de esto, necesitamos una forma de enviar y recibir datos desde nuestros dispositivos inteligentes. Estas señales son recibidas y transmitidas por los modems que obtenemos por los proveedores de internet, estos tienen un velocidad de transferencia, el cual se refiere a la cantidad de datos que pueden transitar en el modem por cada cierto tiempo y que son regulados por los proveedores. Actualmente existen los routers, que no solo reciben y envian señales, sino que tambien pueden gestionar los datos que pasan por nuestra dirección IP, deciden donde iran nuestros datos, que datos y por donde.




Podemos facilitar el proceso de identificacion en la red, adquiriendo un dominio. Un dominio es identificador, un sistema de nombres que nos permiten traducir la direccion IP, a caracteres más cortos y memorizables, lo cual permite una busqueda más facil de nuestra dirección. Los dominios se caracterizan por que poseen un nombre y al final terminan con ciertas siglas que significan, que vamos a hacer con nuestro lugar en internet; Un ejemplo de esto:


nombre.com (.com es el dominio más famoso)
nombre.com.co (.com.co da a entender que la dirección IP proviene especificamente de colombia)


Mediante la dirección IP o por medio de un dominio, podemos conocer los datos o información que hemos subido a internet, pero ser una parte muy diminuta del internet no es suficiente, pues necesitamos subir estos archivos a internet, y que estos puedan ser accedidos por cualquier persona, en cualquier momento. por lo tanto, necesitamos un Host.




Un Host es un servicio que proveen algunas compañias, las cuales nos proveen el acceso a subir nuestros datos o información a un servidor, el cual se va a encargar de mantener nuestros datos todo el tiempo en internet, para que cualquier persona pueda acceder a estos. Ya tenemos nuestra dirección IP, la cual nos va a identificar en internet, tenemos un Host, que va a tomar nuestra dirección IP y va a subir mediante servidores nuestros archivos a internet y los va a tener todo el tiempo conectados para la accesibilidad de cualquier persona. Ahora, ¿como encontramos nuestros datos especificos dentro de nuestra dirección IP o dominio?.


El Host nos provee un Link. Un link es la forma de acceder a elementos especificos dentro de nuestra dirección IP, todas las páginas web tienen un link que permiten un facil acceso a los datos que buscamos, que a su vez estan siendo subidos a ciertos servidores, con sus respectivos dominios.


https://www.youtube.com/ (Dominio)
https://www.youtube.com/watch?v=dQw4w9WgXcQ (Link de contenido especifico)



Ya tenemos la dirección exacta de nuestra página web, pero ahora…



¿Como ingresamos a internet?



         Hay distintos metodos para acceder a internet, pues existen los navegadores web, populares programas que nos permiten navegar por este oceano infinito de información de una forma organizada e intuitiva, pues el navegador nos pregunta que deseeamos investigar o preguntar y este nos genera millones de resultados a partir de la busqueda que realizamos, esta busqueda es otra forma de encontrar páginas web, pues el navegador obtiene la información de las páginas web y segun palabras clave, o contenido especifico, la busqueda de la informacion especifica que buscamos puede ser exitosa o no.




Existen varios metodos para realizar busquedas más existosas en diferentes navegadores, por medio de caracteres especificos para seleccionar ciertas caracteristicas en la información, estos caracteres o tipo de busqueda se llaman operadores de busqueda, algunos de los más utiles podrían ser (estos operadores funcionan en Google, el navegador web mas grande actualmente):


define:Google (Buscar definiciones)
filetype:pdf Google (Busca tipos de archivos especificos)
@Google (Busca redes relacionadas con la busqueda)
allintext:Google (Busca páginas con la palabra especifica)
related:Google (Busca páginas relacionadas con la busqueda)




Ahora conoces lo basico acerca de internet, todo lo que conlleva el simple hecho de visitar una página web, crear una, la necesidad de un proveedor de internet, como hacernos un pequeño lugar dentro del oceano infinito que nos facilita el internet y hacernos de la forma más sencilla con el conocimiento de toda la humanidad.
