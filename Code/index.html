TeShare es una empresa la cual busca facilitar y popularizar toda información esencial referente al hardware y software de computadores, en esta pagina web se explicara de una forma sencilla los componentes que hacen funcionar nuestros computadores, de igual forma se encontrara información útil para el consumidor sobre las aplicaciones Fl Studio, Filmora e Illustrator y por ultimo se verán las principales funciones de Google y Word, junto con los fundamentos de una pagina web y una aplicación.
