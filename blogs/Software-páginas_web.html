¿Que es “Internet”?


         El Internet o Network; término que procede de “red” e “Interconnect” (en la etimologia de la palabra en ingles), por lo que podemos pensar este termino como una gran red que genera una conexion entre usuarios y paginas web a nivel global.


El internet nació para facilitar la comunicación y fuente de información al rededor del mundo, que por consecuente, generó grandes cambios a nivel social, pues se convirtió en el metodo más veloz y eficiente de comunicación personal o masiva de información. De igual manera, este se a convertido en un elemento impresindible en la educación, pues por primera vez en la historia de la humanidad, tenemos todo el conocimiento al alcance de nuestras manos, ¡es más accesible que nunca!


La interconexion global que mecioné anteriormente, surge apartir de una transferencia de hipertextos (enlaces o direcciones que conectan páginas web entre sí) de usuarios a otros por medio de diferentes tipos de estructuras que permiten organizar la información textual o de multimedia, dentro de pequeñas paginas web y a su vez, estas puedan ser compartidas en esta gran red, internet.




¿Como estructuramos esta información?


         Para que una página web pueda presentar la informacion que queremos comunicar, debemos organizarla para que esta sea accesible y consumible, por lo cual, necesitamos un sistema que nos facilite este proceso, lo cual genero el nacimiento de “HTML”.


El HTML (HyperText Markup Language) es un lenguaje que nos permite estructurar nuestra información en una página web, y a su vez, por medio del hipertexto conectar nuestra página web con el internet.


Este lenguaje funciona por medio de etiquetas, las cuales el navegador detecta y luego muestra la información en pantalla, este lenguaje nos permite crear una serie de configuraciones para generar una forma eficiente y atractiva de mostrar esta información; un ejemplo de estas etiquetas podria ser:



<"html">...<"/html"> (Determina el inicio y final de la página web)
<"title">...<"/title"> (Etiqueta utilizada para dar nombre a la página web)
<"img src=”/...”"> (Etiquea utilizada para agregar imagenes)


Este lenguaje es el estandar para la estructuracion de una página web, pero este proceso no queda solo ahí, pues además de estructurar y organizar, podemos decorar cada una de las partes que componen la página web, mediante otros lenguajes como el CSS y no solo eso, tambien podemos convertir nuestra simple pagina web en una aplicacion en si, es decir, que en vez de solo presentar la información para ser consumida, podemos generar una interaccion de la persona que observa la página y la página en si, por medio de diferentes acciones que permiten una funcionalidad adicional a la página, por lenguajes como JavaScript, php, Python.

Ahora que conocemos que es internet, como estructuramos, decoramos y generamos una funcionalidad en el contenido para esta red global…
